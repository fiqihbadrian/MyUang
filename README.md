## 🌟 Fitur Utama

### 💰 Pencatatan Transaksi Lengkap
- ✅ Pemasukan, Pengeluaran, dan Transfer
- ✅ Kalkulator built-in untuk perhitungan cepat
- ✅ Format rupiah otomatis (Rp 1.000.000)
- ✅ Kategori yang dapat disesuaikan

### 👛 Multi-Wallet Management
- ✅ Kelola berbagai akun/dompet
- ✅ Customize icon dan warna untuk setiap wallet
- ✅ Track saldo real-time
- ✅ Transfer antar wallet

### 📊 Analitik & Laporan
- ✅ Grafik donut interaktif
- ✅ Filter: Harian, Bulanan, Tahunan
- ✅ Trend per kategori dengan line chart
- ✅ Export ke Excel (.xlsx)
- ✅ Klik kategori untuk lihat detail trend

### 🎨 Kategori Fleksibel
- ✅ Customize kategori sesuai kebutuhan
- ✅ Icon dan warna untuk setiap kategori
- ✅ Grouping otomatis (Income/Expense/Transfer)
- ✅ Edit dan hapus kategori

### 💾 Backup & Restore
- ✅ Export data ke JSON
- ✅ Import data kapan saja
- ✅ Data aman tersimpan lokal

### 🔒 Privasi & Keamanan
- ✅ **100% Offline** - tidak perlu internet
- ✅ Data tersimpan lokal di perangkat
- ✅ Tidak ada tracking atau analytics
- ✅ Tidak ada iklan
- ✅ Tidak perlu registrasi/login
- ✅ **GRATIS selamanya!**

### Cara Install

📖 **[Baca Instruksi Install Lengkap](INSTALL_INSTRUCTIONS.md)**

**Quick Steps:**
1. Download APK dari [Releases](https://github.com/fiqihbadrian/MyUang/releases)
2. Enable "Install Unknown Apps" di Settings
3. Install APK
4. Bypass Play Protect warning (tap "Install Anyway")
5. Done! ✅

⚠️ **Play Protect Warning = NORMAL!** Ini karena app tidak dari Play Store. App 100% aman, source code bisa dicek di repo ini.

## 🛠️ Tech Stack

- **Framework:** Flutter 3.x
- **Language:** Dart
- **Database:** Hive (Local NoSQL)
- **Packages:**
  - `hive` & `hive_flutter` - Local storage
  - `fl_chart` - Interactive charts
  - `intl` - Date & number formatting
  - `excel` - Excel export
  - `share_plus` - File sharing

## 🔐 Privacy & Security

MyUang sangat serius soal privasi:

✅ **No Data Collection** - Aplikasi TIDAK mengumpulkan data apapun
✅ **No Server Connection** - 100% offline, tidak ada koneksi ke server
✅ **No Analytics** - Tidak ada tracking atau analytics
✅ **No Permissions** - Tidak minta akses lokasi, kontak, kamera, dll
✅ **Local Storage Only** - Semua data tersimpan di device kamu
✅ **Open Source** - Source code bisa di-audit
